# Adventure Blog 

## Project Description
Adventure Blog is a simple web page that showcases beautiful travel destinations and outdoor experiences. The blog highlights nature, landscapes, and adventure photography to inspire exploration and travel.


## Technologies Used
- HTML5
- Git & GitHub for version control


## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your web browser
